# 云端打包指引（GitHub Actions）

本项目已配置好 GitHub Actions，push 到仓库后自动在云端并行编译三大平台安装包。**不需要本机装 Rust、不需要 Mac、不需要 Linux。**

## 打包出来的产物

| 平台 | 文件 | 说明 |
|---|---|---|
| Windows | `*.msi` | Windows Installer（企业分发） |
| Windows | `*.exe` | NSIS 安装器（个人用户友好） |
| macOS | `*-arm64.dmg` | Apple Silicon（M 系列芯片） |
| macOS | `*-x86_64.dmg` | Intel Mac |
| Linux | `*.deb` | Debian / Ubuntu 安装包 |
| Linux | `*.AppImage` | 免安装可执行文件（所有发行版） |

---

## 第一次使用：把项目推到 GitHub

```bash
# 1. 在 GitHub 上新建一个空仓库（比如 voice-demo）

# 2. 本地初始化 + 提交
cd I:\voiceDemo
git init
git add .
git commit -m "Initial commit: Tauri + sherpa-onnx SenseVoice"

# 3. 关联远程仓库并推送
git remote add origin https://github.com/你的用户名/voice-demo.git
git branch -M main
git push -u origin main
```

推送完成后，去 GitHub 仓库的 **Actions** 标签页看进度，大约 **15-25 分钟**完成三平台构建。

---

## 触发构建的三种方式

| 方式 | 触发条件 | 用途 |
|---|---|---|
| 自动 | `git push` 到 main/master | 日常开发 |
| 自动 | `git push origin v1.0.0` (打 tag) | **正式发版，自动创建 GitHub Release** |
| 手动 | GitHub Actions 页面 → "Build All Platforms" → Run workflow | 想临时打一次包 |

## 下载安装包

### 方式 A：从 Actions Artifacts（每次构建都有，保留 90 天）

1. 打开 GitHub 仓库 → **Actions** 标签
2. 点最近一次构建
3. 页面底部 **Artifacts** 区域，下载：
   - `VoiceDemo-Windows-MSI`
   - `VoiceDemo-Windows-NSIS`
   - `VoiceDemo-macOS-arm64`
   - `VoiceDemo-macOS-x86_64`
   - `VoiceDemo-Linux-deb`
   - `VoiceDemo-Linux-AppImage`

### 方式 B：从 Release（打 tag 后永久保存）

```bash
git tag v1.0.0
git push origin v1.0.0
```

构建完成后自动出现在 GitHub 仓库的 **Releases** 页面，任何人可以直接下载。

## Workflow 关键设计

- **模型自动下载**：workflow 会从 sherpa-onnx GitHub Releases 下 `model.int8.onnx`（~230MB）+ `tokens.txt`，用 GitHub Actions cache 加速，同一 key 只下一次
- **Rust 缓存**：`swatinem/rust-cache` 缓存 `~/.cargo` 和 `target/`，二次构建从 20 分钟降到 5-8 分钟
- **Linux 用 ubuntu-22.04**：因为 Tauri 1.x 只支持 webkit2gtk 4.0，Ubuntu 24.04 只有 4.1
- **macOS 双架构**：`macos-14` 出 arm64，`macos-13` 出 x86_64
- **失败不互相拖累**：`fail-fast: false`，一个平台挂了另外两个继续跑

## 已知问题 / 后续可选优化

1. **macOS 未签名**：DMG 用户首次打开会提示"未验证的开发者"。要签名需要 Apple Developer 账号（$99/年）+ 在 workflow 里加 secrets。目前用户可以右键 → 打开来绕过
2. **Windows 未签名**：NSIS/MSI 会有 SmartScreen 警告。签名需要 EV 代码签名证书
3. **包体积大**：模型 230MB + 运行时 ~50MB，成品安装包 ~300MB。如果不想内嵌模型，可以改成首次启动时下载

## 本机开发（可选）

如果之后你的网络能装 Rust 了：

```bash
# 1. 装 Rust（需要能访问 static.rust-lang.org）
rustup default stable

# 2. 下模型
.\download-model.ps1

# 3. 开发
npm install
npm run tauri dev

# 4. 本机打 Windows 包
npm run tauri build
# 产物在 src-tauri/target/release/bundle/
```
